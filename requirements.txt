# Hostel Room Allocation System - Requirements
#
# This project intentionally uses ONLY Python's standard library:
#   - tkinter   (GUI)      -> built into Python on Windows/Mac
#                              (Linux users may need: sudo apt-get install python3-tk)
#   - sqlite3   (Database) -> built into Python on all platforms
#
# There is nothing to install via pip. This file is kept for completeness
# and so graders/reviewers can see at a glance that no external packages
# are required.
#
# Minimum Python version: 3.8+
